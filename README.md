# CSV 自動加工・横結合アプリ (Browser-based)

複数の電力データCSVファイルを、ブラウザ上で一括リネーム・横結合するアプリケーションです。
サーバー処理を必要とせず、すべてのデータ処理がクライアントサイド（ブラウザ内）で完結するため、高速かつ安全に動作します。

## 機能

1.  **一括読み込み**: 10ファイル以上のCSVをドラッグ＆ドロップで一括読み込み。
2.  **自動リネーム**: 
    *   対象: F列（Index 5）以降、またはE列（Index 4）以降のデータ列
    *   ルール: `1-7-1_PCS 有効電力量(kWh)` のような形式を `pcs_1-7-1_kwh` に自動変換。
3.  **横結合 (Outer Join)**:
    *   A列（日付）をキーとしてデータを統合。
    *   B, C, D列は自動的に無視。
    *   日付が欠損している行は空欄として処理（データ欠損を防ぎます）。
4.  **CSV出力**: 結合されたデータを `merged.csv` としてダウンロード。

## 技術スタック

*   **HTML5 / CSS3**: UI構造
*   **Tailwind CSS**: スタイリング
*   **JavaScript (ES6+)**: メインロジック
*   **PapaParse**: 高速CSV解析ライブラリ
*   **FontAwesome**: アイコン

## 使用方法

1.  ブラウザで `index.html` を開きます。
2.  画面中央のエリアに、結合したいCSVファイルをすべてドロップします。
3.  「処理実行」ボタンをクリックします。
4.  ログエリアで処理経過を確認します。
5.  処理が完了すると表示される「CSVダウンロード」ボタンを押して、`merged.csv` を保存します。

## データ仕様上の注意

*   **入力ファイル**: A列が日付であることが必須です。
*   **文字コード**: UTF-8を推奨しますが、主要なエンコーディングは自動判別される場合があります。文字化けする場合はCSVをUTF-8で保存し直してください。
*   **行の整合性**: ファイルごとに行数（日付の範囲）が異なっていても、全ファイルに含まれるユニークな日付をすべて網羅して結合します。

## 開発者向け情報

*   サーバーサイドプログラム（Python/Node.js）は使用していません。
*   `js/app.js` 内の `renameRegex` 変数を変更することで、リネーム規則をカスタマイズ可能です。
